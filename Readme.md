## 💡 Sobre o Projeto

Criação do Proffy, uma plataforma para conexão professores e alunos desenvolvido durante a NLW da Rocketseat.

## 🚀 Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- ReactJS
- React Native
- TypeScript
- HTML5 e CSS3
- NodeJS

## 📥 Executar esse projeto no seu computador

- Clonar Repositório: `git clone https://github.com/guilhermetj/Proffy.git`

Ir para a pasta 'web' para rodar o front-end, ou ir para pasta 'server' para rodar o back-end:

- Instalar dependências: `yarn install`
- Rodar Aplicação: `yarn start`
